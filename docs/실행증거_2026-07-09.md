## `/api/backtest` 응답 (HTTP 200)

백엔드 실행: 저장소 루트에서 `.venv/Scripts/python -m CBS.app` (Flask :5000).

```json
{
  "data_file": "BTC_1h_2020-01-01_2021-01-10.csv",
  "end_value": -123761232.41,
  "lost": 0,
  "return_pct": -12476.12,
  "start_value": 1000000,
  "strategy": "RSIStrategy",
  "total_trades": 1,
  "won": 1
}
```

RSIStrategy로 포함된 BTC 1h CSV(2020-01-01~2021-01-10)를 백테스트한 결과다. 수익률 -12476.12%는 2024년 원본 전략 로직(`CBS/Trade/test.py`)이 그대로 낸 값이며, 전략 로직은 이번 작업에서 동결 대상이라 수치 타당성은 검증하지 않았다.

## `npm run build` 결과 요약

`front/` 에서 `npm run build` 실행 — `Compiled with warnings.` (에러 없음). 경고는 전부 이번 작업이 건드리지 않은 기존 파일들의 ESLint `no-unused-vars` / `react/jsx-pascal-case` 규칙 위반(BacktestPage.js, DryrunPage.js, IntroPage.js, MainNewsPage.js, ResultPage.js, StrategyPage.js, Testst1.js, BackGround.js). `build/` 산출물 정상 생성, gzip 후 `main.js` 305.27 kB.

## 백테스트 차트

`CBS.Trade.main.run_backtest(plot_path=...)`로 candlestick 차트 저장을 시도했으나, 프로세스가 CPU를 거의 소모하지 않는 상태(수 분간 누적 CPU 5초 미만)로 응답 없이 멈춰 강제 종료했다. backtrader 1.9와 신형 matplotlib(3.11) 간 플로팅 경로 마찰로 판단되며, CLI 차트는 이번 증거에 포함하지 못했다. 이 한계는 README §상태에도 명시한다.
